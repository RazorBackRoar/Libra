# L!bra

L!bra is a precision-engineered macOS application built for focus, speed, and intelligent workflow execution.

## Features

- Fast, focused desktop workflow support
- Python-based codebase with tests
- macOS-oriented development setup

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/RazorBackRoar/Libra.git
cd Libra
```

2. Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install dependencies and run tests:

```bash
python3 -m pip install -U pip
pytest
```
